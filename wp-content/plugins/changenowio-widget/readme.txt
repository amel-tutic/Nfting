== Description ==

Instant non-custodial limitless cryptocurrency trading platform - exchange Bitcoin to Ethereum, Ripple, Bitcoin Cash, XMR at the best rates, no account needed! Buy BTC with USD and EUR in a few simple steps!

== Installation ==

1. Upload the folder "changenowio-widget" to "/wp-content/plugins/"
2. Activate the plugin through the "Plugins" menu in WordPress
3. Set up API Key in "Settings" -> "ChangeNow.io Widget"
4. Use the widget "Changenow.io Exchange" in "Appearance" -> "Widgets" or just place the shortcode [changenowio_exchange_widget] where necessary.
